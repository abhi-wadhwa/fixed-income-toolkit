# fixed-income-toolkit

Yield curve bootstrapping, bond pricing, duration/convexity risk metrics, Nelson-Siegel-Svensson fitting, and scenario analysis.

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/fixed-income-toolkit.git
cd fixed-income-toolkit
make install
make run
make test
```

See `docs/` for the math.
